# Sales Performance Dashboard

## Overview
This project is a sales dashboard built using Tableau to analyze business performance across regions, categories, and time.

## Tools Used
- Tableau
- CSV dataset (Superstore data)

## Key Insights
- Sales performance varies by region
- Certain product categories generate higher revenue
- Monthly trends show changes in sales over time

## Dashboard Features
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Interactive filtering
